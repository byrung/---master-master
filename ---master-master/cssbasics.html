CSS 선택자 종류

1. 전체 선택자 (*)   2. 태그 선택자 (태그)   3. 아이디 선택자 (#아이디)
4. 클래스 선택자 (.클래스)   5. 후손 선택자 (header h1)   6. 자손 선택자 (header > h1)

기본 선택자

1.전체 선택자 : HTML 페이지 내부의 태그 모두 선택
2.태그 선택자 : HTML 페이지 내부의 특정 태그 모두 선택
h1 { color:red; } 3.아이디 선택자 : 특정 id가 있는 태그 선택 (중복X)
#header {color:blue} 4.클래스 선택자 : 특정 클래스가 있는 태그 선택 .header {color:green}
