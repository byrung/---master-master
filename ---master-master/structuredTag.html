1.입력양식태그

사용자에게 정보를 입력받는 요소

대표적으로 get(직접 전달) post(별도의 방법으로 전달(비밀스러움))이 있다.


입력 양식 태그 종류

form 입력 양식의 시작과 끝 표시

input 태그

1. text 글자 입력 양식   2. button 버튼
3. checkbox 체크 박스   4. file 파일 입력 양식
5. hidden 해당 내용 표시 안 함   6. image 이미지 형태 생성
7. password 비밀번호양식   8. radio 라디오 버튼
9. reset 초기화 버튼   10. submit 제출

textarea 태그

cols/rows 여러 행의 글자 입력 양식 생성, cols는 너비 rows는 높이

select, optgroup, option 태그

선택 양식 생성, 옵션 그룹화, 옵션 생성

fieldset, legend 태그

입력 양식의 그룹 지정, 입력 양식 그룹의 이름 지정

2.공간 분할 태그

웹페이지의 레이아웃 구성을 위해 공간 분류, 공간을 분류해야 css로 원하는 레이아웃 구성 가능

공간 분할 태그 종류

1. div - 블록 형식으로 공간 분할

2. span - 인라인 형식으로 공간 분할

시맨틱 태그

특정 태그에 의미부여

header - 머리말, nav - 하이퍼링크들을 모아 둔 내비게이션, aside - 본문 흐름에 벗어나는 노트나 팁
section - 문서의 장이나 절에 해당하는 내용, article - 본문과 독립적인 콘텐츠 영역, footer - 꼬리말
